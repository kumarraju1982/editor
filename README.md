# 3D BUILDING Service

Bootstrapped with DIGIT_STRICT defaults.

## Quick Start
1. Copy `.env.example` to `.env`.
2. Install dependencies: `npm install`
3. Run tests: `npm test`
4. Start dev service: `npm run dev`

## Implemented module
- `building-plans` end-to-end module with validation, in-memory repository, state transitions, and API tests.
